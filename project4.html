Resume Screening Platform using LLMs

Automated, scalable resume parsing and scoring using Large Language Models (LLMs) for efficient recruitment and candidate-job matching.

Project Overview

This project presents an end-to-end platform for resume screening and analysis powered by Large Language Models (LLMs). The system automates the extraction of key features from resumes, parses them into structured formats, and scores candidates based on job descriptions, enabling faster and more objective recruitment processes.

Workflow & Methodology
1. Resume Conversion & Parsing

Uploaded resumes are first converted into plain text format for uniform processing. LLMs are then used to extract relevant features (such as education, skills, experience) and generate a structured JSON representation of each resume.

2. Resume Template Generation

The parsed JSON data is utilized to automatically generate standardized resume templates. This ensures consistency and readability across all candidate profiles, facilitating easier comparison and review.

3. Resume Scoring & Matching

Each resume is scored against a given job description using LLM-based semantic matching. The platform computes a weighted average score reflecting the candidate's fit for the role, considering both required and preferred qualifications.

4. Asynchronous Processing & Summarization

To handle multiple resumes and job descriptions efficiently, the platform employs asynchronous parsing and summarization. This allows for rapid, scalable analysis and real-time feedback to recruiters or hiring managers.

Key Features
  • Automated extraction and parsing of resume data using LLMs
  • Standardized JSON resume templates for consistency
  • Semantic scoring and ranking of candidates based on job descriptions
  • Weighted average scoring for nuanced candidate-job matching
  • Asynchronous processing for high throughput and scalability
  • Summarization of job descriptions for rapid screening
Repository Structure
  • convert_to_txt.py: Converts uploaded resumes to plain text
  • extract_features_llm.py: Uses LLMs to extract and structure resume data
  • generate_resume_template.py: Creates standardized resume templates from JSON
  • score_resume.py: Scores resumes against job descriptions
  • weighted_average.py: Calculates weighted average score for candidates
  • async_resume_parser.py: Asynchronous resume parsing module
  • async_jd_summarizer.py: Asynchronous job description summarization
  • README.md: Project overview and usage instructions
How to Use
  • Convert resumes to text using convert_to_txt.py
  • Extract features and generate JSON using extract_features_llm.py
  • Create resume templates with generate_resume_template.py
  • Score resumes against job descriptions using score_resume.py
  • Use asynchronous modules for batch processing
  • Refer to README.md for detailed setup and instructions
